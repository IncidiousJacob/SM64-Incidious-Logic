# SM64-Incidious-Logic
Tired of questionable movement logic? Tired of only having 2 progressive keys in your pool? Tired of people not knowing why you call it long gump? look no further, this apworld is for you
To setup: 
Step 1. follow the regular setup guide for sm64-ap up to the compile build step
Step 2. Click the Box that says "Show Advanced Option"
Step 3. Click "compile Custom Build"
Step 4. Replace Github repository with https://github.com/MuratDev41/sm64ex-ap-custom
Step 5. Continue building your launcher according to the official sm64-ap setup guide
Step 6. Good Luck and have fun
